# 图片合成应用

这是一个基于 Next.js 的图片处理应用，允许用户上传两张图片并通过 Coze API 生成新的合成图片。

## 功能需求

- 用户可以同时上传两张图片
- 支持图片预览功能
- 调用 Coze API 处理图片
- 显示并允许下载生成的新图片
- 支持常见图片格式 (JPG, PNG)

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- axios (用于 API 请求)

## 页面风格

- 苹果官方网站风格样式
- 使用 Coze API 的合成效果
